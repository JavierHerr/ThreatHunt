Executive Summary

This threat hunt was conducted to identify suspicious PowerShell execution across enterprise endpoints. The hunt focused on encoded commands, unusual parent-child process relationships, and suspicious outbound network connections. Several suspicious events were identified, but no confirmed active compromise was found within the scoped environment.

Objective

Identify evidence of malicious PowerShell execution, possible payload delivery, and follow-on activity such as persistence or outbound communication.

Hypothesis

Adversaries may be using PowerShell with encoded commands to download and execute payloads while evading traditional signature-based detections.

Scope

  • Timeframe: March 1, 2026 – March 7, 2026
  • Assets: 2,400 Windows endpoints and 120 servers
  • Users: Corporate user accounts in production
  • Environment: Enterprise Windows environment
  • Data Sources: EDR, Sysmon, DNS logs, proxy logs, SIEM telemetry

Methodology

  • Queried endpoint telemetry for encoded PowerShell execution
  • Reviewed abnormal process lineage such as Office spawning PowerShell
  • Correlated process execution with outbound network connections
  • Reviewed persistence mechanisms including scheduled tasks and registry run keys

Key Findings

Finding 1: Encoded PowerShell Execution on WS-1044

Severity: Medium

Confidence: Moderate

Affected Asset: WS-1044

PowerShell was launched from WINWORD.EXE with an encoded command. This was followed by an outbound HTTPS connection to a rare domain not commonly observed in the environment.

Assessment: Suspicious behavior consistent with a phishing-driven execution chain.

Status: Investigated, no persistence confirmed.

Finding 2: Rare External Domain Contact

Severity: Medium

Confidence: Moderate

Affected Asset: WS-1044

Proxy and DNS logs showed communication to a rare external domain shortly after suspicious PowerShell activity.

Assessment: Possible staging or payload retrieval attempt.

Status: Domain blocked for review.

MITRE ATT&CK Mapping

  • T1059.001: PowerShell
  • T1105: Ingress Tool Transfer
  • T1071.001: Web Protocols

Impact Assessment

Risk Level: Medium

Suspicious execution behavior was identified on at least one endpoint. No confirmed lateral movement or data exfiltration was observed within the available telemetry. Risk remains moderate due to limited certainty about initial payload execution intent.

Actions Taken

  • Reviewed affected endpoint telemetry
  • Isolated suspicious host for analysis
  • Blocked rare external domain
  • Initiated detection review for Office-to-PowerShell execution

Recommendations

  • Enable PowerShell Script Block Logging across all endpoints
  • Tune alerts for Office spawning PowerShell
  • Expand review of similar rare outbound domains
  • Perform a follow-up hunt for persistence artifacts

Limitations

  • Some endpoints had incomplete command-line logging
  • DNS retention limited long-term visibility
  • Scope did not include all legacy systems

Conclusion

The threat hunt identified suspicious PowerShell-related activity consistent with possible phishing-based execution. No confirmed enterprise-wide compromise was identified within the defined scope. Additional hardening, visibility improvements, and focused follow-up hunting are recommended.
